# Exos - Public Scripts

Dieses Repository enthält öffentliche Lua Scripts.

## Struktur

- `public.lua` - Dein öffentliches Script

### Raw URL verwenden

Um dein Script als Raw-Datei zu laden, verwende:
```
https://raw.githubusercontent.com/thherealone/Exos/main/public.lua
```

Für private Scripts siehe das `Exoss` Repository.
